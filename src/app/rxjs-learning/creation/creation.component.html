🏥 Doctor Appointment Clinic — build the morning schedule step by step

Problem: A cardiology clinic opens every morning. The receptionist needs to publish today’s bookable doctor slots on the appointment board.

What we built: A clinic dashboard that loads doctors from doctorsData.json. Clicking Open Morning Clinic runs an RxJS pipeline that builds the schedule step by step.

fromEvent() — waits for the button click; a real user action must happen before anything runs.

of() — emits 3 fixed clinic details (name, shift, date) that we already know upfront.

from() — streams each available doctor one-by-one to scan and verify their slot.

toArray() — collects all scanned doctors into one final schedule array shown on the board.

When to use each creation operator
OperatorWhat it createsIn this clinic example
fromEvent(el, 'click') Observable from a DOM event Receptionist clicks "Open Morning Clinic" — starts the workflow
of(a, b, c) Observable that emits fixed values synchronously Clinic name, shift time, and date — values you already know
from(array) Observable that emits each array item one-by-one Scan each available doctor slot individually before confirming
toArray() Collects all emissions into one array when source completes Build the final morning schedule array for the appointment board
