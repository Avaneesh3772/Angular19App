📊 Executive Business Dashboard — combine multiple streams

Problem: An executive dashboard must load multiple datasets, react to live filters, pair records by index, and show interleaved alerts from different departments.

What we built: A dashboard hub using existing mock JSON files. Each accordion demonstrates one combination operator with a distinct real-world pattern.

forkJoin() — waits for all HTTP calls to finish before emitting once (like Promise.all()).

combineLatest() — emits whenever any input changes; always uses the latest value from each stream.

zip() — pairs values by index: employee[0] with artist[0], employee[1] with artist[1], etc.

merge() — merges multiple streams and emits values as they arrive, interleaved.
