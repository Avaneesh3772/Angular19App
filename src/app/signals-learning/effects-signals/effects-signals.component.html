effect()

🎨 Theme & Font Preferences — run side effects when signals change

effect(() => { ... }) — runs immediately on creation, then re-runs automatically whenever any signal it reads changes. Use for side effects: localStorage, DOM updates, logging, analytics — never for computing values (use computed() for that).
