🏥 Doctor Directory — use HTTP Observables as Signals

doctors = toSignal(observable$, { initialValue: [] }) — converts an Observable into a read-only Signal. Auto-unsubscribes when the component is destroyed. initialValue is returned until the first emission arrives. Always catchError before passing to toSignal() to prevent uncaught errors from crashing the component.
