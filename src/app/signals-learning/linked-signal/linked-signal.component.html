🏢 Department Selector — writable derived signal that auto-resets

linkedSignal(() => derive from source) — like computed() but writable. When source signals change → it resets to the derived value. When manually .set() → stays overridden until source changes again.
computed()
  • Read-only — cannot call .set()
  • Always reflects derived value
  • Use for pure derived state
linkedSignal() ← this demo
  • Writable — supports .set() and .update()
  • Derives initial value from source signals
  • Resets to derived value when sources change
  • Override stays until source changes
